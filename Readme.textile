h1. XML to JSON

Transforms strings of XML to JSON using a simple XSLT.

h2. Usage

<pre>
require 'xml_to_json'

puts XmlToJson.transform(<<-XML)
<items>
  <item>
    <id>1</id>
    <name>Foo</name>
  </item>
  <item>
    <id>2</id>
    <name>Bar</name>
  </item>
</items>
XML
</pre>

Transforms into:

<pre>
{"items":[
  {
    "id":1,
    "name":"Foo"
  },
  {
    "id":2,
    "name":"Bar"
  }
]}
</pre>

Or:

<pre>
require 'xml_to_json/string'

xml.to_json
</pre>
